Mouse

GlobalEventHandlers.onmousemove;

Dieses event wird abgefeuert wenn der benutzer die maus innerhalb des selektierten elementes bewegt, das kann das ganze dokument sein, aber auch einzelne elemente.

  • element.addEventListener('mousemove', (event) => {})
  • element.onmousemove = (event) => {}

GlobalEventHandlers.onmouseover;

Dieses event wird abgefeuert wenn der benutzer die maus über ein element bewegt. Der unterschied zwischen .onmousemove; und .onmouseover ist das .mouseover; nur einmal abgefeuert wird, wenn das element "gehovert" wird, während das event bei .onmousemove; konstant ausgelöst wird;

  • element.addEventListener('mouseover', (event) => {})
  • element.onmouseover = (event) => {}

GlobalEventHandlers.onmouseout;

Dieses event wird abgefeuert wenn der benutzer ein element mit der maus verlässt.

  • element.addEventListener('mouseout', (event) => {})
  • element.onmouseout = (event) => {}

GlobalEventHandlers.onmousedown;

Dieses event wird abgefeuert wenn wir eine maustaste runterdrücken.

  • element.addEventListener('mousedown', (event) => {})
  • element.onmousedown = (event) => {}
