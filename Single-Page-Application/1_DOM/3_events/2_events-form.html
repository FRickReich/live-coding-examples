GlobalEventHandlers.onfocus;

Das event .onfocus; feuert ein callback wenn ein element in den focus gerät, also zum beispiel ein input element das angeklickt wurde.

  • element.addEventListener('focus', (event) => {})
  • element.onfocus = (event) => {}

GlobalEventHandlers.onblur;

Dieses event wird ausgelöst wenn wir den focus von einem element verlieren, zum beispiel wenn wir ein input feld verlassen.

  • element.addEventListener('blur', (event) => {})
  • element.onblur = (event) => {}

GlobalEventHandlers.onsubmit;

Das submit event wird abgefeuert wenn ein formular gesendet, also submitted wird.

  • element.addEventListener('submit', (event) => {})
  • element.onsubmit = (event) => {}

Das formular wurde abgeschickt!

GlobalEventHandlers.onchange;

onchange wird immer dann abgefeuert, wenn die werte eines inputs vom benutzer verändert werden, zum beispiel wenn der benutzer etwas in ein textfeld eingibt.

  • element.addEventListener('change', (event) => {})
  • element.onchange = (event) => {}

GlobalEventHandlers.onclick;

Das event wird abgefeuert wenn der benutzer auf ein element klickt.

  • element.addEventListener('click', (event) => {})
  • element.onchange = (onclick) => {}
